Place the following screenshot files in this folder:

| File | Screenshot to take |
|------|--------------------|
| `app-overview.png` | Full application window (any tab open) |
| `tab-contests.png` | Contests tab |
| `tab-import.png` | Import Logs tab |
| `tab-crosscheck.png` | Cross-Check tab |
| `tab-scoring.png` | Scoring tab |
| `tab-leaderboard.png` | Leaderboard tab with results loaded |

PNG format preferred.
